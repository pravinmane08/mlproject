# 🎓 Student Performance Prediction System

A full-stack **Machine Learning Web Application** that predicts student math performance and provides interactive analytics through a dashboard.

---

## 📌 Project Overview

This project uses **Machine Learning + Flask + Data Visualization** to:

- Predict student math scores
- Store predictions in a database
- Visualize performance using charts
- Provide insights into student performance

---

## 🚀 Features

### 🧠 Prediction System
- Predicts math score using ML model
- Takes input like:
  - Gender
  - Reading Score
  - Writing Score
  - Parental Education
  - Lunch Type
  - Test Preparation

---

### 📊 Analytics Dashboard
- 📈 Prediction Trend (Line Chart)
- 📊 Score Distribution (Histogram)
- 🥧 Performance Breakdown (Pie Chart)
- 📋 Prediction History Table

---

### 🧠 AI Insights
- Excellent Performance 🎉
- Average Performance 👍
- Needs Improvement ⚠️

---

### 🗑 Data Management
- Store predictions in SQLite database
- Clear entire history with one click

---
